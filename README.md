# TODO App

## Description
TODO App ek simple task management application hai jisme users apne tasks ko add, view aur mark as done kar sakte hain.

## Features
- Add tasks with a title and description
- View a list of tasks
- Mark tasks as done

## Technologies Used
- HTML
- CSS
- JavaScript
