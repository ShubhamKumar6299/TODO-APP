# TODO App

## Description
The TODO App is a simple task management application where users can add, view, and mark their tasks as done.

## Features
- Add tasks with a title and description
- View a list of tasks
- Mark tasks as done

## Technologies Used
- HTML
- CSS
- JavaScript
